Antwoorden van opdracht 1 tot 5:

De pagina heeft zelf een default stylesheet. De elementen in de HTML document hebben dus zelf een style gekregen als default.

Opdracht 1:

De waarden width en height worden genegeerd wanneer de elementen weergegeven worden als inline elementen.
De waarden worden als woorden gezet wanneer je inline-block en inline gebruikt.

Opdracht 2:

De margins komen van de default stylesheet van de div element.

Opdracht 3:

De tekst staat horizontaal gecentreerd omdat we in elke div de property text-align voor center hebben gekozen. De tekst van elk element in de div zal in het horizontaal gecentreerd staan.
De property vertical-align zal hier niet werken omdat het text-align hier wordt toegepast op de tekst in de div elementen maar de vertical-align wordt toegepast op inline elementen.
Vertical align bepaald hoe de tekst uitgelijnd wordt bijvoorbeeld een macht. Je kan 2 tot de macht 3 hebben dan kan je deze 3 een vertical align property toekennen met een waarde super. De 3 zal wat meer naar boven zijn uitgelijnd dan de 2 die op zijn baseline staat.
