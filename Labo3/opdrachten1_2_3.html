De accordion section bestaat uit een header met een titel en een icoontje en eronder tekst. De titel staat in een h3, het icoontje in een span element in die h3 element en de tekst staat in een p element die in een div element zit.
Telkens als je op een gesloten section klikt wordt de klasse veranderd van de h3 element en van de div element.
In de HTML document wordt er verwezen in de script tag naar een website met code waarin de accordion zich moet gedragen maar in de DOM tree wordt die code uitgescreven als attributen.
