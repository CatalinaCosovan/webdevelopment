Eenvoudige input en output

Opdracht 1

De return value van de confirm functie als de gebruiker op een van de buttons klikt is een boolean (retourneert waar of onwaar).
De return value van de prompt functie als de gebruiker een tekst intypt en op 'ok' klikt is een string (tekst).
De return value van de prompt functie als de gebruiker op 'cancel' klikt is void (niks).
