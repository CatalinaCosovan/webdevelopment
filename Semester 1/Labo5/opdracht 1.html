De siblings van het <ul> element zijn: p element en h1 element.
De ancestors van het tweede <li> element zijn: ul element, body element en html element.
De descendants van het <body> element zijn: h1 element, p element en ul element met daarin de li elementen.
De enige element in een geldig HTML document die geen parent hebben is de HTML element.
Elk element kan meerdere child elementen hebben.
